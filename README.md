# DSA-Problem-Tracker

A simple Java console-based application to track Data Structures & Algorithms practice.

## Features
- Add DSA problems with topic and status
- View all added problems
- Mark problems as solved
- Menu-driven console interface

## Tech Stack
- Java
- OOP concepts
- ArrayList
- Console I/O

## Purpose
Built as a mini project to strengthen Java fundamentals, OOP, and problem-solving skills.
